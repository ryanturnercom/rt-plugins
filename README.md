# rt-plugins

A Claude Code plugin marketplace featuring `rt-voice` - a plugin that plays themed audio sounds for Claude Code hook events.

## Author

**Ryan Turner** - [ryanturner.com](https://ryanturner.com)

## What's Included

### rt-voice

Plays sound effects when Claude Code triggers hook events like SessionStart, ToolUse, Notifications, and more. Supports custom themes with randomized sound selection.

**Features:**
- 10 hook events supported
- Custom sound themes
- Volume control via slash commands
- Cross-platform audio playback (pygame)

**Slash Commands:**
- `/create-theme <name>` - Create a new theme folder
- `/change-theme` - Switch between themes
- `/volume-up` / `/volume-down` - Adjust volume

See [rt-voice/README.md](rt-voice/README.md) for detailed usage.

## Installation

Clone this repository and configure Claude Code to use the plugin marketplace.
